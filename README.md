# Foundation

Система автоматизации производственного предприятия по выпуску косметической продукции.

## Описание проекта

Foundation — это веб-приложение для управления основными бизнес-процессами производственного предприятия. Система позволяет вести учет сырья, контролировать производство продукции, управлять складом, фиксировать продажи, отслеживать бюджет и формировать отчеты.

Проект разработан с использованием Next.js, TypeScript и Microsoft SQL Server.

---

## Основные возможности

### Авторизация пользователей

* Вход в систему по учетным данным
* Разграничение прав доступа по ролям
* Защита маршрутов через middleware

### Производство

* Создание продукции
* Учет произведенных товаров
* Контроль количества готовой продукции
* Работа с рецептурами

### Закупка сырья

* Добавление закупок
* История закупок
* Автоматическое обновление складских остатков

### Склад

* Учет сырья и материалов
* Просмотр текущих остатков
* Добавление и удаление материалов

### Продажи

* Продажа готовой продукции
* История продаж
* Контроль остатков товаров

### Рецептуры

* Управление составом продукции
* Привязка ингредиентов к товарам
* Расчет расхода сырья

### Бюджет

* Пополнение бюджета
* Контроль расходов
* Отслеживание финансового состояния предприятия

### Отчеты

* Финансовая отчетность
* Аналитическая информация по работе предприятия

## Установка проекта

### Клонирование репозитория

```bash
git clone https://github.com/MustapakulovAzamat/foundation.git
cd foundation
```

### Установка зависимостей

```bash
npm install
```

### Настройка переменных окружения

Создайте файл `.env.local`

```env
DB_USER=your_user
DB_PASSWORD=your_password
DB_SERVER=localhost
DB_DATABASE=FoundationDB
```

## Запуск проекта

Режим разработки:

```bash
npm run dev
```

Открыть в браузере:

```text
http://localhost:3000
```

## Архитектура системы

Система построена по клиент-серверной архитектуре.

Пользователь взаимодействует с интерфейсом Next.js, который обращается к API-маршрутам. API выполняет бизнес-логику и взаимодействует с Microsoft SQL Server для хранения и получения данных.

```text
Пользователь
      ↓
Интерфейс Next.js
      ↓
API Routes
      ↓
SQL Server
```
